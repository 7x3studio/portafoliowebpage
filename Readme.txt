Portfolio Project - Migue Obregon

Description:
This portfolio showcases my work as a graphic designer. 
It highlights my skills in visual storytelling, typography, color theory, 
branding, and modern design aesthetics, including dark mode and Apple-inspired minimalism.


Project Structure:
- index.html      -> Main landing page
- css/            -> CSS files including main styles and theme variables
- js/             -> JavaScript files for interactivity and scroll effects
- images/         -> All images used in the portfolio
- read.txt        -> This file, providing an overview of the project

Features:
- Modern, dark-themed design
- Apple-inspired fonts and UI
- Responsive layout for desktop, tablet, and mobile
- Smooth scroll and scroll-triggered animations
- Reusable components: buttons, cards, sections
- Social links integrated in the footer
